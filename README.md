# El Buen Sabor - Restaurante Italiano

Sitio web para un restaurante italiano ficticio, desarrollado como proyecto de aprendizaje de HTML y CSS.

## Descripción

Este proyecto es una página web completa para un restaurante que incluye:
- Página de inicio con imagen hero
- Sección "Nosotros" con historia del restaurante
- Menú organizado por categorías (entradas, platos principales, postres)
- Galería de imágenes
- Información de ubicación y horarios
- Pie de página con redes sociales

## Tecnologías Utilizadas

- HTML5 semántico
- CSS3
  - Flexbox para layouts
  - CSS Grid para galería
  - Variables CSS (custom properties)
  - Media queries para diseño responsive
- Google Fonts (Playfair Display + Lato)
